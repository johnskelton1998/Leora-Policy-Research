## ðŸ’¡ FAQ

**Q: How much will workers comp cost?**
A: $1,100-$2,400 annually for 3 employees. Texas Mutual often lowest net cost after 30% dividend. Pay-as-you-go options run $85-140/month.

**Q: Can we use a broker or should we buy direct?**
A: Both work. Brokers provide comparison shopping at no cost (commission paid by carrier). Recommended: Get quotes both ways.

**Q: What if an employee reports an injury?**
A: 1) Provide immediate care if needed, 2) Report to supervisor same day, 3) Complete incident report within 24 hrs, 4) Employer reports to carrier, 5) Employee sees designated provider, 6) Modified duty offered if possible.

**Q: Can employees refuse unsafe work?**
A: Yes. If hazards can't be immediately corrected, employee should leave, notify supervisor, and document. No discipline for good-faith safety refusal.

**Q: What's the cell phone policy?**
A: **Zero tolerance.** No texting, emailing, social media, or handheld calls while driving. Must pull over safely to use phone. Violation may result in termination.
